
## Getting Started

```bash
git clone
cd
npm install
```

Add the .env file (be aware that this file is not tracked by git) and add the following:

```bash
ALCHEMY_API_KEY=SOME-STRING-OF-CHARS
INFURA_API_KEY=SOME-STRING-OF-CHARS
```
Then, run the development server:

```bash
npm start dev
```
